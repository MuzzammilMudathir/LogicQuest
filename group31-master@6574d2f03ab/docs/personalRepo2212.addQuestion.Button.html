Attributes: text (str): Text displayed on the button. x, y (int): X and Y position of the button. width, height (int): Width and height of the button. action (callable): Function to execute on button click. Methods: draw(win): Draws the button on the specified window. clicked(pos): Checks if the button is clicked at the given mouse position.
